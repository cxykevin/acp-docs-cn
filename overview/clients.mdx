以下客户端可以与 ACP 代理一起使用：

- [Agmente](https://agmente.halliharp.com) (iOS)
- [AionUi](https://github.com/iOfficeAI/AionUi)
- [aizen](https://aizen.win)
- [DeepChat](https://github.com/ThinkInAIXYZ/deepchat)
- DuckDB
  - 通过 [sidequery/duckdb-acp](https://github.com/sidequery/duckdb-acp) 扩展
- Emacs 通过 [agent-shell.el](https://github.com/xenodium/agent-shell)
- [JetBrains](https://www.jetbrains.com/help/ai-assistant/acp.html)
- [marimo notebook](https://github.com/marimo-team/marimo)
- [neovim](https://neovim.io)
  - 通过 [CodeCompanion](https://github.com/olimorris/codecompanion.nvim) 插件
  - 通过 [carlos-algms/agentic.nvim](https://github.com/carlos-algms/agentic.nvim) 插件
  - 通过 [yetone/avante.nvim](https://github.com/yetone/avante.nvim) 插件
- [Obsidian](https://obsidian.md)
  - 通过 [Agent Client](https://github.com/RAIT-09/obsidian-agent-client) 插件
- [Sidequery _(即将推出)_](https://sidequery.dev)
- [Tidewave](https://tidewave.ai/)
- [Toad](https://www.batrachian.ai/)
- [带有 AI SDK 的 Web 浏览器](https://github.com/mcpc-tech/ai-elements-remix-template) (由 [@mcpc/acp-ai-provider](https://github.com/mcpc-tech/mcpc/tree/main/packages/acp-ai-provider) 提供支持)
- [Zed](https://zed.dev/docs/ai/external-agents)
